    Replace the scaffold-generated schema with the complete schema. Define these tables:

    1. **users** (pgTable "user"):
       - id: text PK
       - name: text nullable
       - email: text not null
       - emailVerified: timestamp (mode: "date") nullable
       - image: text nullable

    2. **accounts** (pgTable "account") -- required by @auth/drizzle-adapter:
       - userId: text not null, references users.id cascade
       - type: text not null
       - provider: text not null
       - providerAccountId: text not null
       - refresh_token: text nullable
       - access_token: text nullable
       - expires_at: integer nullable
       - token_type: text nullable
       - scope: text nullable
       - id_token: text nullable
       - session_state: text nullable
       - PK: composite (provider, providerAccountId)
       - Index on userId

    3. **sessions** (pgTable "session") -- required by @auth/drizzle-adapter even with JWT:
       - sessionToken: text PK
       - userId: text not null, references users.id cascade
       - expires: timestamp not null

    4. **verificationTokens** (pgTable "verificationToken") -- required by @auth/drizzle-adapter:
       - identifier: text not null
       - token: text not null
       - expires: timestamp not null
       - PK: composite (identifier, token)

    5. **bases** (pgTable "base"):
       - id: text PK with $defaultFn(() => crypto.randomUUID())
       - name: text not null
       - userId: text not null, references users.id cascade
       - createdAt: timestamp defaultNow not null
       - updatedAt: timestamp defaultNow not null
       - Index on userId

    6. **tables** (pgTable "table"):
       - id: text PK with $defaultFn(() => crypto.randomUUID())
       - name: text not null
       - baseId: text not null, references bases.id cascade
       - createdAt: timestamp defaultNow not null
       - Index on baseId

    7. **columns** (pgTable "column"):
       - id: text PK with $defaultFn(() => crypto.randomUUID())
       - tableId: text not null, references tables.id cascade
       - name: text not null
       - type: text enum ["text", "number"] default "text" not null
       - order: integer default 0 not null
       - Index on tableId

    8. **rows** (pgTable "row"):
       - id: text PK with $defaultFn(() => crypto.randomUUID())
       - tableId: text not null, references tables.id cascade
       - rowOrder: integer default 0 not null
       - cells: jsonb.$type<Record<string, string | number | null>>() default {} not null
       - createdAt: timestamp defaultNow not null
       - CRITICAL INDEX: composite on (tableId, rowOrder, id) named "row_tableId_rowOrder_id_idx" -- this is the keyset cursor pagination index

    9. **views** (pgTable "view"):
       - id: text PK with $defaultFn(() => crypto.randomUUID())
       - tableId: text not null, references tables.id cascade
       - name: text not null
       - config: jsonb.$type<{ filters: unknown[]; sorts: unknown[]; hiddenColumns: string[]; searchQuery: string }>() default { filters: [], sorts: [], hiddenColumns: [], searchQuery: "" } not null
       - Index on tableId

    Export Drizzle `relations` for each table so relational queries work.

    This pushes schema directly to Supabase (uses DIRECT_URL from drizzle.config.ts).

    IMPORTANT: Use the exact Auth.js adapter table structure from @auth/drizzle-adapter docs. The adapter expects specific column names (provider, providerAccountId, etc.). Do NOT rename them.

    2. Create `src/server/auth.ts` (full instance with adapter + JWT):
       ```typescript
       import NextAuth from "next-auth";
       import { DrizzleAdapter } from "@auth/drizzle-adapter";
       import { db } from "~/server/db";
       import { authConfig } from "./auth.config";

       export const { auth, handlers, signIn, signOut } = NextAuth({
         ...authConfig,
         adapter: DrizzleAdapter(db),
         session: { strategy: "jwt" },
         callbacks: {
           session({ session, token }) {
             if (token.sub) session.user.id = token.sub;
             return session;
           },
           jwt({ token, user }) {
             if (user) token.sub = user.id;
             return token;
           },
         },
       });
       ```

    3. Update `src/proxy.ts` to import ONLY auth.config (NOT auth.ts):

       ```typescript
       import { authConfig } from "~/server/auth.config";

       export const { auth: proxy } = NextAuth(authConfig);
       export default proxy;

       export const config = {
         matcher: ["/((?!api|_next/static|_next/image|favicon.ico).*)"],
       };
       ```
