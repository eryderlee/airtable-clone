3. Add `view.delete` mutation for deletion. On success: invalidate views list. If the deleted view was the active view, redirect to the first remaining view:
```tsx
const deleteView = api.view.delete.useMutation({
  onSuccess: async (deleted) => {
    await utils.view.getByTableId.invalidate({ tableId });
    if (deleted.id === activeViewId) {
      // Fetch updated list and redirect to first view
      const remaining = await utils.view.getByTableId.fetch({ tableId });
      if (remaining.length > 0 && remaining[0]) {
        router.push(`/base/${baseId}/${tableId}/view/${remaining[0].id}`);
      }
    }
  },
});
```
