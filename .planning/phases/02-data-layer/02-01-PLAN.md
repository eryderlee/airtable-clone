Create `src/server/api/routers/base.ts` with these procedures (all protectedProcedure):

**base.getAll** — query: Select all bases where `bases.userId === ctx.session.user.id`. Return array.

**base.create** — mutation: Input `{ name: z.string().min(1).max(255) }`. Insert with `userId: ctx.session.user.id`. Return created base.

**base.update** — mutation: Input `{ id: z.string().uuid(), name: z.string().min(1).max(255) }`. Update where `id = input.id AND userId = ctx.session.user.id`. If no rows affected, throw NOT_FOUND. Set `updatedAt: new Date()`. Return updated base.

**base.delete** — mutation: Input `{ id: z.string().uuid() }`. Delete where `id = input.id AND userId = ctx.session.user.id`. If no rows affected, throw NOT_FOUND. Return deleted base.

Create `src/server/api/routers/table.ts` with these procedures (all protectedProcedure):

**table.getByBaseId** — query: Input `{ baseId: z.string().uuid() }`. First verify ownership: select from `bases` where `id = input.baseId AND userId = ctx.session.user.id`. If not found, throw NOT_FOUND. Then select all tables where `baseId = input.baseId`, ordered by `createdAt asc`.

**table.create** — mutation: Input `{ baseId: z.string().uuid(), name: z.string().min(1).max(255), seed: z.boolean().default(true) }`.
1. Verify base ownership (same check as getByBaseId).
2. Insert table.
3. If `seed` is true:
   a. Create 3 default columns: "Name" (text, order 0), "Notes" (text, order 1), "Status" (text, order 2).
   b. Create 5 rows with faker data: cells keyed by actual column IDs. Name = `faker.person.fullName()`, Notes = `faker.lorem.sentence()`, Status = `faker.helpers.arrayElement(["Todo", "In Progress", "Done"])`. rowOrder = index (0-4).
   c. Create one default view named "Grid View" with default config `{ filters: [], sorts: [], hiddenColumns: [], searchQuery: "" }`.
4. Return the created table.

CRITICAL schema notes (the research doc has some inaccuracies vs actual schema):
- Use `createTable` imported from `~/server/db/schema` (tables are prefixed with `airtable_`). Actually, import the table constants (`bases`, `tables`, `columns`, `rows`, `views`) directly from schema — do NOT call `createTable` in routers.
- IDs are `text` type, not `uuid` type — but they hold UUID values, so `z.string().uuid()` is correct for validation.
- Column position field is named `order`, not `position`.
- Views use a single `config` JSONB column (not separate `filters`/`sorts`/`hiddenColumns`/`searchQuery` columns).
- Use dynamic import for faker: `const { faker } = await import("@faker-js/faker")` to avoid bundling in edge runtime.
- Import `eq`, `and`, `asc` from `drizzle-orm`.
- Import `TRPCError` from `@trpc/server`.

**table.update** — mutation: Input `{ id: z.string().uuid(), name: z.string().min(1).max(255) }`. Verify ownership via join: `tables INNER JOIN bases ON tables.baseId = bases.id WHERE tables.id = input.id AND bases.userId = ctx.session.user.id`. If not found, throw NOT_FOUND. Update name. Return updated table.

**table.delete** — mutation: Input `{ id: z.string().uuid() }`. Same ownership check via join. Delete the table. Return deleted table.

Create `src/server/api/routers/column.ts` with these procedures (all protectedProcedure):

**column.getByTableId** — query: Input `{ tableId: z.string().uuid() }`. Verify ownership via join: `tables INNER JOIN bases ON tables.baseId = bases.id WHERE tables.id = input.tableId AND bases.userId = ctx.session.user.id`. If not found, throw NOT_FOUND. Select all columns where `tableId = input.tableId`, ordered by `order asc`.

**column.create** — mutation: Input `{ tableId: z.string().uuid(), name: z.string().min(1).max(255), type: z.enum(["text", "number"]).default("text") }`.
1. Verify table ownership (same join pattern).
2. Get max `order` value from existing columns for this table: `SELECT MAX(order) FROM columns WHERE tableId = input.tableId`. If no columns exist, use -1.
3. Insert column with `order: maxOrder + 1`.
4. Return created column.

**column.update** — mutation: Input `{ id: z.string().uuid(), name: z.string().min(1).max(255) }`. Verify ownership via 3-level join: `columns INNER JOIN tables ON columns.tableId = tables.id INNER JOIN bases ON tables.baseId = bases.id WHERE columns.id = input.id AND bases.userId = ctx.session.user.id`. If not found, throw NOT_FOUND. Update name. Return updated column.

**column.delete** — mutation: Input `{ id: z.string().uuid() }`. Same 3-level ownership check. Delete the column. Return deleted column. Note: cell data cleanup (removing the column key from JSONB cells) is NOT needed here — the column simply stops being referenced. The JSONB cells retain stale keys harmlessly.

Create `src/server/api/routers/view.ts` with these procedures (all protectedProcedure):

**view.getByTableId** — query: Input `{ tableId: z.string().uuid() }`. Verify table ownership (same 2-level join as column router). Select all views where `tableId = input.tableId`.

**view.create** — mutation: Input `{ tableId: z.string().uuid(), name: z.string().min(1).max(255) }`. Verify table ownership. Insert view with default config `{ filters: [], sorts: [], hiddenColumns: [], searchQuery: "" }`. Return created view.

**view.update** — mutation: Input `{ id: z.string().uuid(), name: z.string().min(1).max(255) }`. Verify ownership via 3-level join: `views INNER JOIN tables ON views.tableId = tables.id INNER JOIN bases ON tables.baseId = bases.id WHERE views.id = input.id AND bases.userId = ctx.session.user.id`. If not found, throw NOT_FOUND. Update name. Return updated view.

**view.updateConfig** — mutation: Input `{ id: z.string().uuid(), config: z.object({ filters: z.array(z.any()).optional(), sorts: z.array(z.any()).optional(), hiddenColumns: z.array(z.string().uuid()).optional(), searchQuery: z.string().optional() }) }`. Same 3-level ownership check. Merge the provided config fields into the existing config: read the current view, spread existing config, overlay provided fields. Update the `config` column. Return updated view.

NOTE on updateConfig: Use partial merge — only overwrite fields that are provided. Read the current view's config first, then spread: `{ ...currentView.config, ...Object.fromEntries(Object.entries(input.config).filter(([_, v]) => v !== undefined)) }`.

**view.delete** — mutation: Input `{ id: z.string().uuid() }`. Same 3-level ownership check. Delete view. Return deleted view.
